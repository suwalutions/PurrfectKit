# PR Title Guidelines (Lulu-approved)

## Use the following prefixes:

| Type          | Prefix           | Example                                          |
| ------------- | ---------------- | ------------------------------------------------ |
| Fix           | `lulu: bug`      | `[lulu: bug] (#123): fix login crash`              |
|               |                  | `[lulu: bug]: fix typo in footer`                  |
| Chore         | `lulu: chore`    | `[lulu: chore] (#124): update scripts`             |
|               |                  | `[lulu: chore]: update npm scripts`                |
| Dependencies  | `lulu: deps`     | `[lulu: deps] (#125): upgrade packages`            |
|               |                  | `[lulu: deps]: upgrade packages`                   |
| Docker        | `lulu: docker`   | `[lulu: docker] (#126): update container`          |
|               |                  | `[lulu: docker]: rebuild image`                    |
| Documentation | `lulu: docs`     | `[lulu: docs] (#127): add README section`          |
|               |                  | `[lulu: docs]: update CONTRIBUTING.md`             |
| Feature       | `lulu: feature`  | `[lulu: feature] (#128): add search functionality` |
|               |                  | `[lulu: feature]: add search functionality`        |
| Refactor      | `lulu: refactor` | `[lulu: refactor] (#129): improve API service`     |
|               |                  | `[lulu: refactor]: cleanup code structure`         |
| Tests         | `lulu: tests`    | `[lulu: tests] (#130): add unit tests`             |
|               |                  | `[lulu: tests]: add unit tests`                    |
