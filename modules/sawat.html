Malet¶

The purrfectmeow.Malet class specializes in text extraction, supporting various file types like PDFs, images, spreadsheets, and Markdown files. It provides a flexible loader() method with multiple loader options.

See Usage for detailed examples of using purrfectmeow.Malet.

class purrfectmeow.Malet[source]¶

Bases: object

Text extraction class inspired by the Malet Thai cat breed.

This class provides methods to extract text from various file types using specialized loaders. Supported file types include PDFs, images, spreadsheets, and Markdown files. Each loader is optimized for a specific format or extraction technique.

supported_loaders¶

List of available loader names:

  • ‘MARKITDOWN’: MarkItDown for Markdown conversion

  • ‘DOCLING’: Docling for Markdown conversion

  • ‘PYTESSERACT’: Tesseract OCR for image text extraction

  • ‘EASYOCR’: EasyOCR for image text extraction

  • ‘SURYAOCR’: SuryaOCR for image text extraction

  • ‘PYMUPDF’: PyMuPDF for PDF text extraction

  • ‘PYMUPDF_AS_TXT’: Alternate PyMuPDF extraction as plain text

  • ‘PANDAS_EXCEL’: pandas for Excel file extraction

  • ‘PANDAS_CSV’: pandas for CSV file extraction

Type:

list

Examples

>>> from purrfectmeow import Malet
>>> docs = Malet.loader("example.pdf", "example.pdf", loader="PYMUPDF")
static loader(file: BinaryIO, file_name: str, loader: str, **kwargs: Any) → str[source]¶

Load and extract text from a file using the specified loader.

Parameters:

  • file (BinaryIO) – A file-like object opened in binary mode.

  • file_name (str) – Name of the file for reference (used for context or metadata).

  • loader (str) – Loader type to use for extraction.

  • **kwargs (Any) – Additional keyword arguments passed to the loader.

Returns:

Extracted text content, or other data structure depending on the loader.

Return type:

str

Raises:

  • FileNotFoundError – If the file does not exist or cannot be accessed.

  • ValueError – If the specified loader is invalid or unsupported.

Examples

>>> with open("example.pdf", "rb") as f:
...     text = Malet.loader(f, "example.pdf", loader="PYMUPDF")
>>> print(text[:200])  # Prints first 200 characters of extracted text
