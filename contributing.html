Contributing

We welcome contributions to PurrfectKit! To contribute:

  1. Fork the repository on GitHub: https://github.com/suwalutions/PurrfectKit

  2. Create a branch for your feature or bugfix.

  3. Submit a pull request with a clear description of changes.

  4. Report issues or suggest features via GitHub Issues.

Note

Please include tests update documentation for new features.
